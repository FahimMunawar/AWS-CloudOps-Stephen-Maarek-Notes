# 18 — AWS Billing Conductor

## Overview

Customizes how AWS billing data is **displayed and allocated** — does NOT change your actual AWS bill.

---

## Key Capabilities

- Group accounts together
- Apply custom markups and discounts
- Generate **pro forma bills** (custom/internal invoices)

---

## Use Cases

| Audience | Use Case |
|---|---|
| **Enterprises** | Split bill by department or cost center |
| **MSPs** | Apply custom pricing per customer |
| **Finance teams** | Chargeback or showback models |
| **Subsidiaries** | Transparent internal invoice system |

---

## Quick Reference

```
Billing Conductor: customizes billing display/allocation — NOT actual bill
  Group accounts + apply markups/discounts + generate pro forma bills

Use cases: enterprise cost centers, MSP custom pricing, finance chargeback, internal invoicing
Target: large organizations with complex accounting needs
```
